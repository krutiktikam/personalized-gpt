# Aura AI: Next-Gen Evolution Plan (Phase 5 & 6) - April 7, 2026

This plan outlines the transition from a "Smart Assistant" to an "Autonomous Engineering Partner."

## Phase 5: Intelligence & Context (The "Deep Thinker" Update)
*Goal: Move beyond simple memory to active knowledge retrieval and smarter reasoning.*

- [ ] **RAG Integration (Retrieval-Augmented Generation):**
    - Implement a Vector Database (e.g., ChromaDB or FAISS) to store Aura's documentation and the user's past project snippets.
    - Aura will be able to "search" its own history for specific code solutions it gave you weeks ago.
- [ ] **Code-Review Mode:**
    - A new personality mode `/review` where Aura analyzes a provided code snippet for security vulnerabilities and "clean code" violations.
- [ ] **Multi-Model Routing:**
    - Use the 3B model for quick chat and "quirks," but automatically route complex architecture tasks to a larger 7B or 8B model (if hardware/API allows) for better logic.
- [ ] **Tool Use (Function Calling):**
    - Allow Aura to "check the time," "create a local folder," or "check if a file exists" by mapping LLM intent to Python functions.

## Phase 6: Professional Infrastructure (The "SaaS-Ready" Update)
*Goal: Prepare the architecture for multiple users and professional monitoring.*

- [ ] **JWT Authentication:**
    - Secure the FastAPI endpoints with JSON Web Tokens so Aura can eventually support private user accounts.
- [ ] **Database Migration (SQLite -> PostgreSQL):**
    - Move to a more robust database to handle concurrent read/writes and more complex relationship mapping between "Facts" and "Tasks."
- [ ] **Observability Dashboard:**
    - Integrate Prometheus/Grafana or a simple custom dashboard to track:
        - Inference latency (how fast Aura thinks).
        - Token usage.
        - Common user emotions detected over time.
- [ ] **Async Pipeline:**
    - Refactor `run_pipeline` to be fully asynchronous to prevent the API from "hanging" during heavy model inference.

## New Feature Ideas (The "Aura Ecosystem")
1. **"Aura Voice" (TTS/STT):**
   - Integration with Whisper (OpenAI) for speech-to-text and Piper/Coqui for ultra-fast local text-to-speech.
2. **"The Daily Standup":**
   - A proactive feature where Aura sends a summary of your "Pending Tasks" every morning at a set time.
3. **"Shadow Mode":**
   - Aura "thinks" in the background about your previous goals and occasionally interrupts with a "Deep Thought" even if you haven't asked a question.
4. **GitHub Integration:**
   - Allow Aura to read your public GitHub repos directly to give advice based on your *actual* codebases.
